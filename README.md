# SAFER
"SAFER - Safety-Assured Autonomous Flight for Emergency Rescue" develops an end-to-end drone autonomy pipeline that combines localization, motion control, and object detection. The project enables drones to support emergency responders through safer navigation, real-time situational awareness, and faster rescue-oriented surveillance.

This repository is applied on Finnish Drone Hackathon 2 by Ultrahack

See `vision/TRAIN_FIRE_SMOKE.md` for dataset layout and training commands.
